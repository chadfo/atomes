# Atomes studio — site statique

Contenu prêt pour GitHub Pages : déposez tous les fichiers de ce dossier à la racine du dépôt.

1. Créez un dépôt GitHub, uploadez ces fichiers (racine).
2. Settings → Pages → Source : branche main, dossier /(root).
3. Domaine perso : Settings → Pages → Custom domain, puis chez votre registrar
   un enregistrement CNAME "www" → <utilisateur>.github.io (et/ou les A records GitHub pour l'apex).
4. Cochez "Enforce HTTPS" une fois le domaine vérifié.

La page d'accueil est index.html.
